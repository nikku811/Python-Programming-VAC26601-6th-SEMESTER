# EXPERIMENT No. 12: Search Employee by ID

## AIM
- To search employee details using ID.

## SOURCE CODE
```python
emps={101:"Rahul",102:"Aman"}
id=int(input("Enter ID: "))
print(emps.get(id,"Not found"))
```

## OUTPUT
```
Displays employee name or not found.
```

## RESULT
Program executed successfully.
