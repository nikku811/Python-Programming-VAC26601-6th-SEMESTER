# EXPERIMENT No. 13: Sort Employees by Salary

## AIM
- To sort employees based on salary.

## SOURCE CODE
```python
emp={"Rahul":50000,"Aman":40000,"Neha":60000}
sorted_emp=sorted(emp.items(),key=lambda x:x[1])
print(sorted_emp)
```

## OUTPUT
```
Employees sorted by salary.
```
